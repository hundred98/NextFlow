# n8n 移动端应用

一款用于随时随地监控和管理 n8n 工作流的移动应用。

## 功能特性

- 查看和监控您的 n8n 工作流
- 工作流卡片颜色状态指示器：
  - 运行中状态：黄色指示器
  - 活动状态：绿色背景+绿色指示器
  - 已暂停状态：灰色背景+灰色指示器
  - 无法激活状态：灰色背景+灰色指示器
- 查看执行日志和结果
- 快速启用/禁用工作流
- 使用手动触发器、Webhook或表单触发节点手动触发工作流
- 基于位置的触发器
- 相机/相册触发器
- 传感器数据触发器
- 摇晃手势触发器
- 轻量级审批和交互功能

## 会员系统

n8n 移动端应用提供分级会员系统，为不同级别的用户提供差异化的功能和服务：

### 基础用户（免费）
- 基础监控功能：查看工作流状态、执行日志
- 有限的手动触发功能
- 基础通知功能
- 社区支持

### 专业用户（付费）
- 无限制工作流监控
- 高级触发器：位置、传感器、相机等硬件触发
- 完整的手动触发功能
- 优先通知和提醒
- 高级安全功能（应用锁等）
- 优先技术支持

### 团队/企业用户（付费）
- 多用户管理
- 团队协作功能
- 高级安全控制
- 专属客户支持
- 定制化选项

## 技术架构

### 前端开发
- **技术栈**：React Native 跨平台解决方案
- **核心交互设计**：
  - 卡片式工作流列表展示
  - 下拉刷新更新状态
  - 长按菜单快速操作

### 后端适配
- **API 层**：
  - 移动端专用接口
  - 优化数据传输效率
- **节点开发**：
  - 移动端输入节点（位置/图片/录音）
  - 移动端通知节点
  - 扫码触发节点
- **权限控制**：
  - 新增"移动端角色"
  - 限制可访问工作流范围

## 实施路线图

### 第一阶段（MVP+）
- 手动触发功能（Webhook GET/Manual Trigger/Form Trigger/Chat Trigger/Execute Workflow Trigger/Schedule Trigger/Cron Trigger/Interval Trigger/Timer Trigger）
- 工作流列表查看
- 基础状态监控
- 关键日志查看
- 目标：验证核心交互可行性

### 第二阶段（核心场景）
- 硬件触发（位置/扫码）
- 工作流审批功能
- APP内通知推送
- 目标：覆盖核心业务场景

### 第三阶段（体验优化）
- 自定义工作流卡片
- 日志筛选导出
- 离线同步优化
- 多账号切换
- 高级设置选项（生物识别认证、自动登出时间）
- 目标：提升用户体验

## 安全防护措施

1. **数据传输安全**
   - TLS 1.3 加密
   - API 请求签名验证

2. **数据存储安全**
   - 敏感信息系统级加密存储
   - 本地数据分级保护

3. **身份认证**
   - 多因素认证（MFA）
   - 精细化权限管控

4. **设备安全**
   - Root/越狱检测
   - APP 安全加固
   - 自动锁屏保护

5. **操作审计**
   - 关键操作日志
   - 异常行为拦截

6. **应急响应**
   - 远程账号管控
   - 定期安全更新
